# 🚀 Space Tourism Website

## 📌 Sobre o Projeto

Este site foi construído para simular uma agência de turismo espacial multi-página. O desafio consistia em criar uma experiência de usuário fluida, onde o visitante pode explorar destinos, conhecer a tripulação e entender as tecnologias envolvidas, tudo com um design futurista e responsivo.

O projeto foi desenvolvido utilizando o App Router do Next.js, com foco em componentização, responsividade e animações suaves para melhorar a experiência do usuário.

## 🛠️ Tecnologias e Ferramentas

O projeto utiliza o que há de mais moderno no ecossistema de desenvolvimento web:

* **[React.js](https://reactjs.org/):** Biblioteca principal para construção da interface.
* **[Next.js](https://nextjs.org/):** Framework React para produção (Roteamento, SSR e Otimização).
* **[Tailwind CSS](https://tailwindcss.com/):** Framework CSS utilitário para estilização responsiva.
* **[Framer Motion](https://www.framer.com/motion/):** Biblioteca para animações fluidas e transições entre páginas.
* **Google Fonts:** Integração das fontes *Bellefair* e *Barlow Condensed*.

## ✨ Funcionalidades

- [x] **Navegação SPA:** Troca de páginas instantânea sem recarregamento.
- [x] **Exploração de Destinos:** Alternância dinâmica entre Lua, Marte, Europa e Titã.
- [x] **Apresentação da Tripulação:** Slider interativo com informações dos membros.
- [x] **Guia Tecnológico:** Seção detalhada sobre equipamentos espaciais.
- [x] **Responsividade:** Design adaptado para Mobile, Tablet e Desktop.

## ⚙️ Instalação Local

Para rodar este projeto Next.js na sua máquina:

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/GabGStein/space-tourism-site.git

2. **Instale as dependências:**
   ```bash
   npm install

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev

4. **Acesse no navegador:**
   
- Abra http://localhost:3000 para ver o resultado.
